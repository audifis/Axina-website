AXINA Banking API

L'infrastructure de données financières certifiées pour les banques OHADA.

Une API qui connecte directement les établissements bancaires aux données financières certifiées d'une entreprise. Score AXINA. Projection d'absorption. Risque. En un appel.

API Reference

5 endpoints. Tout ce dont une banque a besoin.

Chaque endpoint est sécurisé par consentement. Chaque donnée retournée est traçable depuis le ledger certifié DIP.

POST /consent/share

Créer un token temporaire

Autorise la banque à consulter un dossier client selon un périmètre défini.

GET /bank-ready/company_id

Récupérer le dossier

Retourne score, ratios, états, cashflow et résumé bancaire.

GET /score/company_id

Score financier

Score calculé à partir des données comptables et fiscales disponibles.

GET /cash/projection/90days

Projection cash

Anticipe les tensions de trésorerie et la capacité d'absorption.

GET /risk/company_id

Analyse risque

Identifie anomalies, signaux faibles et zones de vigilance.

Webhooks

Recevez les mises à jour d'un dossier dès qu'un événement financier certifié apparaît.

Sécurité & consentement

Zéro donnée sans consentement actif.

L'entreprise débloque l'accès. Chaque accès à bancaire est enregistré dans un audit log immuable. AXINA ne transforme pas la donnée financière en boîte noire : elle la rend exploitable et vérifiable.
